# Taller de Listas Enlazadas Simples

Este repositorio contiene la solución del taller de listas enlazadas simples en Python.

## Archivos

- `listas_enlazadas.py`: implementación completa de los ejercicios 1 al 10.
- `informe_listas_enlazadas.md`: informe base con investigación, complejidades, pruebas y conclusiones.

## Cómo ejecutar

```bash
python listas_enlazadas.py
```

## Contenido implementado

1. Lista enlazada completa.
2. Eliminar por valor.
3. Insertar por índice.
4. Buscar y contar.
5. Invertir lista.
6. Sumar dos números representados como listas.
7. Intercambiar nodos por parejas.
8. Historial de navegador.
9. Detectar ciclo con Floyd.
10. Partir lista en mitad.

## Benchmark

El archivo incluye una comparación con `timeit` entre:

- `list.insert(0, i)` en listas nativas de Python.
- `prepend(i)` en listas enlazadas.
